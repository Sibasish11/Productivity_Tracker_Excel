# Productivity Tracker Excel Template

A simple yet powerful Excel based **habit and productivity tracker** to help you build consistent habits over a 21 day period (3 weeks). Great for tracking daily goals like reading, exercise, meditation, or any personal productivity tasks.

## Features
- Track up to **11 customizable habits**
- **21 daily tracking columns** organized into Week 1, Week 2, and Week 3
- Automatic date generation from your start date
- Mark completions easily (enter `1`, `x`, or any value)
- Auto-calculated totals, percentages, best/worst habits, and progress summaries
- Pure Excel formulas : no macros, fully compatible with Google Sheets (minor tweaks may be needed)

## How to Use
1. Download the repository.
2. Open `prodTRACKER.xlsx` in Microsoft Excel or compatible software.
3. Enter your **start date** in cell B2 (currently set to January 1, 2026).
4. Replace placeholder habit names ("Article1" etc.) with your own.
5. Mark completed days with any value (e.g., `1` or `x`).
6. Watch totals and progress update automatically!

## Files
- `prodTRACKER.xlsx` : The main tracker spreadsheet.

## Customization
- Add more habits by copying rows and formulas.
- Extend the tracking period by adding columns.
- Enhance with conditional formatting or charts.
